# Blender Drone Assets

This folder contains **3D drone models and assets created in Blender** as part of the SkySpec project.
These models are used as the visual and structural basis for drone simulations across multiple platforms, including Unity and NVIDIA Isaac Sim.

---

## Project Overview

### UAV.blend

This project includes the main **UAV quadcopter model**, designed for simulation and export to different formats such as `.obj` and `.usd`.
The model focuses on providing a clean geometry suitable for simulation workflows rather than photorealistic rendering.
